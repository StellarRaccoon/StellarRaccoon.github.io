Parts of a Computer

Memory

Store temporary data.

  • Used by CPU to store the data for the current processes.
  • Must be fast and easily accessed by CPU.
  • Accessed via Memory Bus
  • Sometimes called "Primary Storage" or "Main Memory"
  • If running lots of applications at once you need lots of Memory

Common Forms of Memory:

  • ROM
  • RAM
  • Registers

Storage

Long term file storage

  • Keep files for extended periods of time
  • Not directly connected to CPU
  • Often accessed via I/O channels (ex: USB port)
  • Often cheap, easy to obtain, and easy to expand
  • If you have lots of applications, videos, or other big files you need lots of Storage

Common Forms of Storage:

  • Hard Disk Drive (HDD)
  • Solid State Drive (SSD)
  • Compact Disk (CD)
  • USB Drives

Central Processing Unit

The brains of the computer.

  • Interpret Programs, Process Data
  • Contains Internal Clock to synch the system
  • Fetches data from input devices
  • Sends proccess result to output or storage.
  • Contains multiple cores which can run in parallel

3 Components of CPU:

  • Control Unit
  • Arithmetic and Logic Unit
  • Registers

Input/Output

Peripherals and interface

  • Collects or displays data
  • Often connected via Universial Serial Bus (USB)
  • Data sent in or out of the CPU
  • Sends proccess result to output or storage.
  • Contains multiple cores which can run in parallel

Common I/O devices:

  • Keyboard
  • Sensor
  • Display Screen
  • Another processes

CPU Components

Control Unit

Processes instructions

  • Sends control signals to other parts of the computer system
  • Decodes instructions
  • Applies instructions

Arithmetic and Logic Unit

Carries Out instructions

  • Completes tasks requested by Control Unit
  • Uses circuits to complete operations

Registers

Small Fast Memory Units

  • Volitile
  • General Purpose and Special Purpose
