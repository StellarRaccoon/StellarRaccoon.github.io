Parts of a Computer

Memory

Store temporary data.
The CPU uses memory to store the data for the current processes. Memory must be fast and easily accessed.
If you run ots of applications at once ypu need lots of Memory
    Common Forms of Memory:
  • ROM
  • RAM
  • Registers

Storage

Long Term File Storage
Storage is where you keep files and save them for an extended period of time.
If you have lots of applications, videos, or other big files you need lots of Storage
Additional storage is often easy to obtain and expand
    Common Types of Storage
  • Hard Disk Drive (HDD)
  • Solid State Drive (SSD)
  • Compact Disk (CD)
  • USB Drives

Central Processing Unit

Runs programs and processes data. The brains of the computer.
If data is collected it fetches data from input devices and stores the result or sends it to output.
It's internal clock synchs the system.
Made up of multiple cores which can run in parallel.
    3 Main Parts
  • Control Unit
  • Arithmetic and Logic Unit
  • registers

Input/Output

Peripherals and Data
Data is often inputted via Universal Serial Bus (USB).
Data outputted by the CPU may be printed, sent to another process, etc.
    Common I/O Devices
  • Keyboard
  • Mouse
  • Display Screen
  • Speakers
